# Caddy + Profit Premium

## Быстрый старт

### Вариант 1: Запуск только Caddy (Next.js уже запущен)

```bash
# Сначала запусти Next.js
npm run start

# Потом в другом окне
start-caddy.bat
```

### Вариант 2: Запуск всего стека одной кнопкой

```bash
start-stack.bat
```

## Что создаёт Caddy

| Внешний порт | Внутренний порт | Куда проксирует |
|-------------|-----------------|-----------------|
| 80 (HTTP)   | 80              | → 3000 (Next.js) |
| 443 (HTTPS) | 443             | → 3000 (Next.js) |

## Твои проброшенные порты (роутер)

| Комментарий   | Локальный IP   | Локальный порт | Протокол | Удалённый порт |
|---------------|----------------|----------------|----------|----------------|
| MoneyServ     | 192.168.0.13   | 5000           | TCP      | 5000           |
| MoneyServProd1| 192.168.0.13   | 443            | TCP+UDP  | 443            |
| MoneyServProd2| 192.168.0.13   | 80             | TCP+UDP  | 80             |

**Важно:** У тебя уже проброшены порты 80 и 443 на 192.168.0.13. Caddy будет слушать эти порты и проксировать на Next.js (порт 3000).

## Установка Caddy (если не установлен)

1. Скачай: https://caddyserver.com/download
2. Положи `caddy.exe` в `C:\Windows\System32\` или в папку проекта
3. Проверь: `caddy version`

## Файлы

| Файл              | Назначение                          |
|-------------------|-------------------------------------|
| `Caddyfile`       | Конфиг Caddy (Caddyfile формат)     |
| `caddy.json`      | Альтернативный конфиг (JSON)        |
| `start-caddy.bat` | Запуск только Caddy                 |
| `start-stack.bat` | Запуск Next.js + Caddy вместе       |

## Доступ к сайту

- **Локально:** http://localhost:3000
- **Из LAN:** http://192.168.0.13 (через Caddy)
- **Из интернета:** http://ТВОЙ_ВНЕШНИЙ_IP (если белый IP)

## Остановка

Просто закрой окно командной строки или нажми `Ctrl+C`.
